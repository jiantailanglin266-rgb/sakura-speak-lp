What's new this round

  • Mini-games completed (4 total) — added Sentence Scramble (tap words into the right order) alongside Word Blitz, Memory Match & Listen It.
  • Avatar — preset selection — replaced the dress-up customizer with a simple picker of ready-made Meemi looks (selection only). These are placeholders, ready to swap for your supplied artwork.
  • Real authentication & accounts — sign-up / login / guest / logout with sessions; the member area is gated (signed-out visitors are redirected to sign in). Email-confirmation flow supported.
  • Member database — member data persists; ready for Supabase (schema & setup guide included).
  • Member management (admin) — the admin console lists members and can suspend / ban / restore and grant or revoke admin — on live data.
  • Guest → account upgrade — guests are nudged to create an account; upgrading keeps the same account & progress.
  • Stripe-ready checkout — the subscribe screen redirects to Stripe (Payment Links) once your links are set; otherwise a demo checkout. No server required.
  • Free / Premium gating — premium features (e.g. mini-games) are locked for free members and unlock after subscribing; the plan status shows in the sidebar.
Everything runs today on the static site in a built-in demo mode (no real charges/data). Setting the Supabase and Stripe details (below) switches the same screens to real auth, database and payments — no code changes.
